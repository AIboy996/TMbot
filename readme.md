# TMBot

给bot发送消息`最近在看 tmdb_id season_num`即可完成影视打卡。

而后bot会把打卡这条消息删掉，然后发送带海报的影视信息。

- 电影打卡
    - 发送：`最近在看 1260951`
    - 自动生成：
    - <img src="assets/2025-01-26-12-29-05.png" width="300">
- 剧集打卡：
    - 发送：`最近在看 46260 2`
    - 自动生成：
    - <img src="assets/2025-01-26-12-30-00.png" width="300">
